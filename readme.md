## Анализатор текста v0.0002 preALFA

Программа позволяет указать в консоли путь к файлу, который нужно проанализировать. Анализирование может происходить на двух языках: русский или английский. В статистику не входят предлогии/местоимения/союзы.
В качестве результат в консоли приложения выводится слово и количество его повторов. Начиная с версии 0.0002 можно анализировать правильность написания скобок, указав при этом путь к папке. Медленными шагами
изучается spring для переноса это в web-пространство.

## Зависимости
Приложение работает на jdk1.8 и выше Gradle - 4.1
